# NLog annotations for Resharper

[![Build status][AppVeyorBadge]][AppVeyor]
[![MIT License][LicenseBadge]](License.md)

This project generate annotations for [NLog](http://nlog-project.org/) to be
used by [JetBrain's ReSharper][R#].

This project uses my [ExternalAnnotationsGenerator][Generator] project to keep
the generation maintainable.
